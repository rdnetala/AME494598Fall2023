Temperature/Humidity

Step-by-Step Guide:

  1. Start AWS Academy lab from AWS Academy Canvas -> Modules -> Sandbox.
  2. Navigate to EC2 and create an instance of a machine on the AWS server. Use the following settings:
    • Linux Ubuntu 20.04
    • t2:micro
    • Continue without key-pair login
    • ~28 Gb storage
  3. Click on the instance link and connect to the EC2 instance via SSH.
  4. Edit the firewall setting to add inbound rules to open certain ports:
    • Custom TCP -> port 1234 -> anywhere IPv4
    • Custom TCP -> port 8080 -> anywhere IPv4
    • If storing data to a database, configure: Custom TCP -> port 27017 -> My IP
  5. Setup the Linux machine on the cloud by installing the required software:
    • Run: curl -s https://gist.githubusercontent.com/tejaswigowda/f289e9bff13d152876e8d4b3281142f8/raw/df37df2e16a3166e686357a045751a7c18bbeebe/installNodeMongoUbuntu16.04.sh | bash
    • Software includes NodeJs, MongoDB, http-server. Update NodeJs to V20 with: sudo npm install && sudo n latest
    • Clone the class/own repository using: git clone
  6. Navigate through the repository and start hosting the folder containing the server.js file. Use the command: node server.js
  7. Open ArduinoIDE, and open the DHT11Default.ino code. Update SSID, password, and server IP to the hotspot login, then upload the Arduino sketch to the microcontroller.
  8. Connect to the EC2 server through the browser to see the data from the microcontroller.
